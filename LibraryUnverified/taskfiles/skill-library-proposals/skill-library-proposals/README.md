# Skill Library Proposals

This bundle turns the full conversation and uploaded project material into a **categorized candidate skill library**.

## What is inside

- one markdown proposal file per skill candidate
- category folders with local README indexes
- `MANIFEST.md` and `manifest.json` for the full registry
- `SOURCES_USED.md` describing where the candidate list came from

## Important note

These are **planning stubs** for the library. They are not finished production `SKILL.md` implementations yet.
