# Model Notes

This directory contains model-specific prompting documentation discovered during scaffolding.

When the `agent-prompt-engineering` skill is used and the chosen model doesn't have notes here, the agent should web-search for that model's prompting best practices and write the findings to a new file in this directory (e.g., `minimax-m2.5.md`, `claude-sonnet-4.md`).

This allows future scaffold runs to reuse model-specific knowledge without repeating the research.
